# Portfolio Web

A responsive portfolio website built with modern HTML, CSS, and JavaScript. This project showcases professional skills, services, projects, and contact information in a clean, mobile-friendly layout.

## Features

- Modular component-based HTML structure
- Responsive design for desktop and mobile
- Smooth navigation and section-based layout
- Dynamic content loaded via JavaScript modules
- Contact form integration and external data retrieval

## Project Structure

- `index.html` — Main landing page
- `assets/` — Images and static media assets
- `components/` — Reusable HTML page sections
- `css/` — Stylesheets for layout and presentation
- `js/` — JavaScript modules for navigation, content retrieval, and contact handling

## Setup

1. Open `index.html` in a browser, or serve the project using a local web server.
2. Verify the site loads correctly on desktop and mobile devices.

## Notes

- Designed for easy customization and content updates.
- Ideal for personal branding and portfolio presentation.
